# Rixit 🃏

Aplicación web inspirada en el juego de mesa Dixit.

## Instalación

```bash
pnpm install # o npm install / yarn install
```

## Desarrollo

```bash
pnpm dev # o npm run dev / yarn dev
```

## Construir para producción

```bash
pnpm build
```

## Tecnologías

- React + TypeScript
- Vite
- Tailwind CSS (modo oscuro por defecto)
